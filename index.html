🔐 OAuth認証設定（オプション）

📋 OAuth認証が必要な場合

  • ユーザーのボード一覧を取得したい
  • REST APIを使用したい
  • ユーザー情報を取得したい
  • 複数ボードにまたがる操作をしたい

🔄 リダイレクトURI設定

# OAuth認証後のリダイレクト先
https://your-domain.com/auth/callback
https://your-domain.com/auth/success
https://your-domain.com/index.html?auth=success

# 複数設定可能（開発用・本番用）
https://localhost:8000/auth/callback
https://your-domain.com/auth/callback

画面上部の To do で「OAuth authentication: Add a publicly reachable redirect URI」と表示されている場合は、上記のリダイレクトURIを追加する必要があります。

🚨 よくあるエラーと対処法

CORS エラーが発生する

適切なHTTPSホスティングを使用し、Miro SDKが正しく読み込まれているか確認してください。

OAuth認証が失敗する

リダイレクトURIが正確に設定されているか、クライアントIDが正しいか確認してください。

アプリが読み込まれない

ブラウザのデベロッパーツールでネットワークエラーを確認し、URLが正しいか検証してください。
